# crear una función sencilla e invocarla
##Sintaxis
````function nombreFuncion(){}````

*	function - Palabra reservada para definir la función. EJEMPLO. function NAME
*	() - Sirve para definir los parámetros que entran dentro de la función.
*	{} - bloque de código o algoritmo, si no la cierras correctamente con "}" te dará un error de sintaxis
*	return - Sirve para retornar o definir la opción principal de la función
##Ejercicio
1. Crear un formulario que recoja con cuatro campos: nombre, email, edad, población
2.	Crear  una función llamada saludar que devuelva un string que salude expandiendo cada una de las variables.
3. 	Tendrá como parámetros el nombre, el email, la edad y población
4. 	mostrará en pantalla un saludo personalizado y su año de nacimiento
